Postman-коллекция содержит 5 запросов.
Все запросы имеют общие переменные BaseURL (кроме 1-го запроса) и Token.
Токен получаем из заголовков запроса и записываем туда же с ключом Authorization.
Перед началом запуска коллекции требуется указать токен.

1 запрос не является проверкой. С помощью него получаем каталог книг, для добавления в корзину.
Содержит в себе переменную отдельную переменную UrlBooks для Url.
После запроса сохраняет id любой книги из каталога в рандомном порядке в переменную IdBook.

2 запрос добавляет товар в корзину.
В теле запроса указывается id книги с 1-го запроса с помощью переменной IdBook.
Содержит проверку на код 200 и проверку на пустое тело ответа.

3 запрос позволяет просмотреть товары, добавленные в корзину.
Содержит проверку на код 200 и проверку на наличие строки "products" в теле ответа.
После запроса сохраняет в переменную Id2Book новую id любой книги из корзины в рандомном порядке. Это требуется, так как для следующих запросов прежний id книг не подходит. Будет использоваться новый id.

4 запрос изменяет количество одного из товаров в корзине.
В теле запроса указывается новый id с помощью переменной Id2Book.
Содержит проверку на код 200 и проверку на наличие строки "products" в теле ответа.

5 запрос удаляет выбранный товар из корзины.
В url указывается новый id книги с помощью переменной Id2Book.
Содержит проверку на код 204 и проверку на пустое тело ответа.
